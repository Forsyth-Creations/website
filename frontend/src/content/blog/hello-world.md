# Hello World

Welcome to the blog! This is where I'll share technical write-ups, project updates, and anything else worth documenting.

## What to Expect

Posts here will cover a range of topics:

- **Robotics** — control systems, swerve drive, simulations
- **Web development** — building things with React, Next.js, and other tools
- **Software engineering** — patterns, tools, and lessons learned

## Adding New Posts

This blog is powered by plain Markdown files. To add a new post, create a `.md` file in `src/content/blog/` with the following frontmatter:

```markdown
---
title: "Your Post Title"
date: "YYYY-MM-DD"
description: "A short summary of the post."
tags: ["tag1", "tag2"]
---

Your content here...
```

The post will automatically appear in the blog index sorted by date.
